Welcome to tweet-rss!

This app requires some setup in order to authenticate it with Twitter. This is necessary as the Twitter API no longer permits un-authenticated requests.

Once this is done, the app will be able to perform its API calls on your behalf, which means it will be able to generate RSS feeds for users whose timeline is private, provided they are following you (just like in the regular client).

This wizard will take you through the steps required to get this app working.
